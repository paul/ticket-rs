# Add .tickets.toml configuration support

Add support for a project-local .tickets.toml config file, discovered by walking upward from cwd. Two keys: ticket_prefix (overrides derived prefix) and ticket_dir (overrides .tickets/ path). Env vars TICKET_PREFIX and TICKET_DIR override the file; TICKETS_DIR remains a legacy alias for TICKET_DIR.
